Habilidades

html

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas da web. É usado para estruturar e apresentar o conteúdo da web, como textos, imagens, vídeos e outros elementos.

css

CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo que é usada para descrever a aparência e formatação de uma página da web. É usado para estilizar elementos HTML como fontes, cores, espaçamentos, entre outros aspectos visuais.

javascript

JavaScript é uma linguagem de programação de alto nível, interpretada e dinâmica, que é amplamente utilizada na web para adicionar interatividade e funcionalidades avançadas às páginas da web.

java

Java é uma linguagem de programação de alto nível, orientada a objetos e de propósito geral. É amplamente utilizada para desenvolver aplicativos para a web, sistemas corporativos, aplicativos móveis, jogos e muito mais. É conhecida por sua portabilidade, segurança e facilidade de uso.

python

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Conhecido por sua sintaxe clara e concisa, o que facilita a leitura e a manutenção do código. É uma linguagem interpretada, ou seja, o código é executado linha por linha, sem a necessidade de compilação prévia.
